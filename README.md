# dpdk_gtp_module
DPDK implementation of GTPv1 user plane gateway.

### Features
- High speed GTP-U packet encapsulation and decapsulation
- Proxy ARP on behalf of UE IPs
- Packet statistics update on stdout
- Node socket aware memory config
